How this was made


The Parts

"Arduino"


Product name: Adafruit Huzzah ESP8266

What does this do?

  • This is a “microcontroller”. It’s a device that are told to do one thing and they do it forever. They are similar to how your cell phone works but instead of being able to run multiple apps at once, they can only run 1 app.

"Distance Sensor"


Product name: Ultrasonic Distance Sensor - 3V or 5V - HC-SR04 compatible - RCWL-1601

What does this do?

  • This allows the arduino to tell how close you are. This is like motion detectors on garage lights.

"LED Lights"


Product name: Neopixel WS8211 Addressable LED Lights

What does this do?

  • These are light strips that each LED on them can be told to be different colors.

"LED Controller"


Product name: FadeCandy

What does this do?

  • TThis tells the lights which individual lights to light up and what color. The computer program sends this data.

"Computer"


Product name: Mac Mini

What does this do?

  • When you program, or code, you provide instructions for the computer to follow. Many programmers write code in text, meaning that they type it out on the keyboard.(Google). On the computer a program is running that listens for the arduinos to say something. It then draws its response on the computer screen and tells the lights to light up.
